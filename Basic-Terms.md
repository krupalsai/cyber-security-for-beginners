# 📖 Basic Cyber Security Terms

A simple glossary of important terms every beginner must know.

---

## A–C

| Term | Meaning |
|------|---------|
| **Antivirus** | Software that detects and removes malicious programs |
| **Authentication** | Verifying who you are (e.g., login with password) |
| **Authorization** | What you are allowed to do after logging in |
| **Backdoor** | A hidden way to access a system, bypassing normal security |
| **Brute Force** | Trying every possible password until one works |
| **CIA Triad** | Confidentiality, Integrity, Availability — the 3 pillars of security |
| **Cipher** | An algorithm used to encrypt or decrypt data |
| **Cookie** | A small file stored in your browser by a website |

## D–F

| Term | Meaning |
|------|---------|
| **DDoS** | Distributed Denial of Service — flooding a server to crash it |
| **Encryption** | Scrambling data so only the right person can read it |
| **Exploit** | Code or technique that takes advantage of a vulnerability |
| **Firewall** | A system that filters incoming and outgoing network traffic |

## H–M

| Term | Meaning |
|------|---------|
| **Hacker (Black Hat)** | A person who breaks into systems illegally |
| **Hacker (White Hat)** | An ethical hacker hired to find and fix vulnerabilities |
| **Honeypot** | A fake system set up to trap hackers |
| **HTTPS** | Secure version of HTTP — data is encrypted in transit |
| **IP Address** | A unique number that identifies a device on a network |
| **Malware** | Malicious software (virus, worm, trojan, ransomware, etc.) |
| **MFA / 2FA** | Multi-Factor Authentication — extra login security step |

## P–Z

| Term | Meaning |
|------|---------|
| **Patch** | An update that fixes a security vulnerability in software |
| **Payload** | The part of malware that does the actual damage |
| **Phishing** | Tricking someone into giving their credentials via fake emails/sites |
| **Port** | A virtual door on a computer for network communication |
| **Ransomware** | Malware that locks your files and demands payment |
| **Social Engineering** | Manipulating people psychologically to gain access |
| **Trojan** | Malware disguised as a legitimate program |
| **VPN** | Virtual Private Network — encrypts your internet connection |
| **Vulnerability** | A weakness in a system that can be exploited |
| **Zero-Day** | A vulnerability unknown to the software vendor |

---
> 💡 Learn these terms well — they appear everywhere in cyber security!
