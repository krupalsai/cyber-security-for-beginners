# 🛡️ Cyber Security for Beginners

Welcome! This repository is a **beginner-friendly** guide to learning cyber security from scratch.  
No prior experience required — just curiosity and dedication.

## 🚀 How to Use This Repo

1. Start with **Roadmap.md** to understand the learning path
2. Read **Basic-Terms.md** to get familiar with the language
3. Study **Networking-Basics.md** — networking is the foundation
4. Learn about **Common-Attacks.md** to understand what you're defending against
5. Practice with the **Tools-List.md** and **Linux-Commands.md**
6. Explore **Resources.md** for free courses and videos

## ⚠️ Disclaimer
