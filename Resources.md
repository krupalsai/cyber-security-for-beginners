# 📚 Free Cyber Security Learning Resources

A curated list of the best **free** resources to learn cyber security.

---

## 🎮 Hands-On Practice Platforms

| Platform | Description | Level |
|----------|-------------|-------|
| [TryHackMe](https://tryhackme.com) | Guided, beginner-friendly hacking labs | Beginner |
| [HackTheBox](https://hackthebox.com) | Real-world hacking challenges | Intermediate |
| [PicoCTF](https://picoctf.org) | CTF for students, great for beginners | Beginner |
| [OverTheWire](https://overthewire.org) | Wargames focusing on Linux & security | Beginner |
| [CyberDefenders](https://cyberdefenders.org) | Blue team / defensive security labs | Intermediate |
| [Pwn.college](https://pwn.college) | Binary exploitation training (ASU) | Advanced |

## 🏆 Certifications (Beginner Roadmap)

```
Start Here:
  └── CompTIA IT Fundamentals (ITF+) — optional
      └── CompTIA Security+         — BEST first cert
          └── CEH (Certified Ethical Hacker) — popular in India
              └── eJPT (eLearnSecurity) — practical hacking
                  └── OSCP — gold standard for pentesters
```
