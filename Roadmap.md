# 🗺️ Cyber Security Learning Roadmap

Follow this step-by-step path to go from **zero to beginner** in cyber security.

---

## Phase 1 — Foundations (Weeks 1–4)

- [ ] Understand what cyber security is and why it matters
- [ ] Learn basic networking (IP, DNS, HTTP, TCP/UDP)
- [ ] Get comfortable with the Linux command line
- [ ] Understand what a server, client, and firewall are

## Phase 2 — Core Concepts (Weeks 5–8)

- [ ] Study common cyber attacks (phishing, malware, SQL injection, etc.)
- [ ] Learn about encryption (symmetric vs asymmetric, HTTPS, SSL/TLS)
- [ ] Understand authentication and authorization
- [ ] Set up a safe lab using VirtualBox + Kali Linux

## Phase 3 — Hands-On Practice (Weeks 9–16)

- [ ] Practice on beginner-friendly platforms: TryHackMe, HackTheBox
- [ ] Learn basic web vulnerabilities (OWASP Top 10)
- [ ] Try simple CTF (Capture The Flag) challenges
- [ ] Understand scanning with Nmap and Wireshark

## Phase 4 — Specialise (Month 4+)

Choose a path that interests you:

| Path | Focus Area |
|------|------------|
| 🔴 Offensive (Red Team) | Ethical hacking, penetration testing |
| 🔵 Defensive (Blue Team) | Monitoring, incident response, SIEM |
| 🟡 Bug Bounty | Finding vulnerabilities in real apps |
| 🟢 GRC | Governance, risk, compliance |

## 📜 Beginner Certifications to Aim For

- **CompTIA Security+** — Best entry-level cert
- **CEH (Certified Ethical Hacker)** — Popular in India
- **Google Cybersecurity Certificate** — Free on Coursera

---
> 💡 Tip: Consistency beats intensity. Study 1 hour every day.
